🎨 Visual Test Reports

📊 About Visual Testing

This page contains visual regression test reports for the SparkEngineWeb project. Reports are generated automatically using Playwright's HTML reporter when visual tests fail in pull requests.

📁 Report Structure

Reports are organized by PR number: /visual-reports/pr-{NUMBER}/

🔗 Access Reports

Visual test reports are linked directly in pull request comments when tests fail. Each report includes:

  • 🖼️ Side-by-side image comparisons
  • 📊 Interactive test timeline
  • 🔍 Zoom functionality for detailed inspection
  • 📋 Complete test execution logs
  • 🎯 Quick navigation to failed tests
